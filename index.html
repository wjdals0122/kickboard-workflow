과제 정의

특수 디스플레이를 어디에, 어떻게 써야 할까?

INPUT
📐

디스플레이 제약

226:775의 극단적 세로 비율 — 일반 서비스에 적용하기 어려운 형태. 단독 서비스보다 기존 인프라와 연계하는 방식이 적합하다고 판단.

💡

핵심 인사이트 발견

세종시-LG전자가 협력하여 전국 100개 이상 설치 중인 PLUSPOT(전동 킥보드 주차 스테이션)을 발견. 이 인프라에 디스플레이를 결합하면 현실적 확장이 가능.

🎯

방향 설정

단순 정보 표시가 아닌, 안전 인식을 바꾸는 서비스를 설계. 강제가 아닌 리워드 기반으로 이용자가 자발적으로 안전 행동을 하도록 유도.

💬

"100% 제거가 목적이 아닌, 예방과 인식 개선에 초점을 맞춘다. 강제가 아닌 즐거움으로."

UX 설계 근거

각 결정이 왜 이 방향으로 이루어졌는가

VALIDATION
①
세로 비율 활용

화면 하단 활용도가 낮아질 수 있는 영역에 광고·날씨 정보 배치. 수익 모델과 정보 제공을 동시에 해결.

②
리워드 기반 설계

강제가 아닌 마일리지·스탬프 미션으로 자발적 안전 행동 유도. 경쟁 심리를 활용한 주간 순위 시스템.

③
4단계 대여 프로세스

킥보드 선택 → 헬멧 인식 → 면허 확인 → 음주 측정(야간). 단계별 안전 확인을 자연스러운 흐름으로 설계.

④
음주 측정 시간 제한

18:00~06:00 야간 시간대에만 음주 측정 적용. 낮 시간대 대여 흐름을 방해하지 않으면서 실질적 위험 시간대를 집중 관리.

💹

경제적 이득

안전 이미지로 탑승 고객 증가 + 디스플레이 광고 수익 창출

이 프로젝트에서 배운 것

제약 → 기회

세로형 특수 디스플레이라는 제약을 단점이 아닌, 킥보드 스테이션이라는 특정 공간에 최적화된 강점으로 전환

현실 가능성

팀장으로서 아이디어의 현실 가능성을 중심으로 방향을 정리하고, 기존 인프라(PLUSPOT)와의 연계로 실현 가능한 서비스 모델 도출

사용자 중심

강제가 아닌 리워드 기반으로 사용자가 자발적으로 안전 행동을 하도록 유도하는 UX 설계 경험
